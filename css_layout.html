Modelo de caja

Esta foto representa el esquema básico del modelo de caja en CSS. Como vemos tenemos diferentes partes: el contenido en HTML; el padding, que es un margen entre el contenido y el borde; el borde, al cual se le pueden aplicar distintos estilos en CSS y delimita el contenido; y el margen, que es el margen entre la caja y otros elementos o cajas del sitio.
